ZIP İndir
Sonraki adımlar:
  1. ZIP dosyasını indirin.
  2. overleaf.com → New Project → Upload Project → ZIP'i seçin.
  3. Menu → Compiler → XeLaTeX seçin.
  4. Recompile → PDF hazır.
Overleaf'e Nasıl Yüklerim?
1
ZIP'i İndirin
"ZIP İndir" butonuna tıklayın ve dosyayı bilgisayarınıza kaydedin. ZIP içinde main.tex, logo ve şekil dosyaları bulunur.
2
Overleaf'e Gidin
overleaf.com adresine gidin. Oturum açın veya ücretsiz hesap oluşturun.
3
Proje Oluşturun
Sol üstten New Project → Upload Project seçin. İndirdiğiniz ZIP dosyasını seçin ve yükleyin.
4
XeLaTeX Seçin ⚠ Kritik
Sol üstte Menu butonuna tıklayın. Açılan panelde Compiler altından XeLaTeX'i seçin. (Varsayılan pdfLaTeX çalışmaz.)
5
Derleyin
Recompile butonuna tıklayın. Birkaç saniye sonra sağ panelde PDF görünür. İstediğiniz değişiklikleri yapıp tekrar derleyebilirsiniz.
6
PDF İndirin
Sağ üstteki Download PDF butonuyla hazır PDF'i bilgisayarınıza indirin.
⚠️ Önemli: Derleyici olarak mutlaka XeLaTeX seçilmelidir. Türkçe karakterler, özel fontlar ve orcidlink paketi yalnızca XeLaTeX ile doğru çalışır. Overleaf varsayılan olarak pdfLaTeX kullanır — bunu değiştirmezseniz derleme hatası alırsınız.
